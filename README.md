## Installation

Clone the repository

```bash
  https://github.com/larasantos8/pokedex-app.git
  cd pokedex-app
```

Install dependencies

```bash
  yarn or npm install
```

Start the Project

```bash
   yarn start or npm start
```

## Running Tests

To run tests, run the following command

```bash
  yarn test or npm run test
```
